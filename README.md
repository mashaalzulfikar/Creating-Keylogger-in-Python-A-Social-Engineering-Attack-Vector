# Creating-Keylogger-in-Python-A-Social-Engineering-Attack-Vector

This repository contains the documentation and source code for an academic assignment regarding Social Engineering and cybersecurity.

## Project Overview

This project provides a practical demonstration of a basic keylogger implemented in Python. The study explores the mechanics of keylogging, how it functions as an attack vector, and the importance of implementing robust cybersecurity countermeasures.

## Files Included

- `report.pdf`: The complete report detailing the theoretical background, literature review, findings, and analysis.


- `keylogger.py`: The Python implementation used for the practical demonstration.

## How to Run

1. **Prerequisites**: Ensure you have Python installed on your system.
2. **Install Dependencies**: This project uses the `pynput` library. Install it via your terminal:
```bash
pip install pynput

```

3. **Execution**: Run the script using the command:
```bash
python keylogger.py

```

4. **Functionality**: Upon execution, the script will monitor keystrokes and log them into a file named `keyfile.txt` in the same directory.
